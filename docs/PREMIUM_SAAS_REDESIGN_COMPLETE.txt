HOVER GLOW EFFECT EXPLANATION:
===============================

The hover glow effect creates a professional, premium feel:

1. Edge Glow (Border):
   - Uses ::before pseudo-element
   - Positioned absolutely with inset: -2px
   - Gradient background (blue shades)
   - mask-composite creates border-only effect
   - Opacity 0 by default, 1 on hover
   - filter: blur(8px) for soft glow

2. Mouse-Following Light:
   - Uses ::after pseudo-element
   - Radial gradient centered at cursor
   - JavaScript tracks mousemove event
   - Calculates position as percentage
   - Updates CSS custom properties
   - Gradient: blue center → transparent edges
   - Very subtle (0.08 opacity)

3. Implementation:
   ```javascript
   const handleMouseMove = (e) => {
     const rect = card.getBoundingClientRect();
     const x = ((e.clientX - rect.left) / rect.width) * 100;
     const y = ((e.clientY - rect.top) / rect.height) * 100;
     card.style.setProperty('--mouse-x', `${x}%`);
     card.style.setProperty('--mouse-y', `${y}%`);
   };
   ```

4. CSS:
   ```css
   .premium-card::after {
     background: radial-gradient(
       circle at var(--mouse-x, 50%) var(--mouse-y, 50%),
       rgba(59, 130, 246, 0.08) 0%,
       transparent 50%
     );
   }
   ```
