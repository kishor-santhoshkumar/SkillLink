HOVER LIGHT EFFECT EXPLANATION:
================================

The hover light effect creates a subtle radial gradient that follows the mouse cursor across cards.

HOW IT WORKS:

1. CSS Custom Properties:
   - Each card tracks mouse position as --mouse-x and --mouse-y percentages
   - These are set dynamically via JavaScript

2. Pseudo-element (::after):
   - Creates an overlay with radial-gradient
   - Gradient center follows cursor: circle at var(--mouse-x) var(--mouse-y)
   - Uses rgba(59, 130, 246, 0.08) for subtle blue light
   - Opacity 0 by default, transitions to 1 on hover

3. JavaScript Mouse Tracking:
   - useEffect hook adds mousemove listener to card
   - Calculates mouse position relative to card bounds
   - Converts to percentage (0-100%)
   - Updates CSS custom properties in real-time

4. Result:
   - Smooth, professional lighting effect
   - Follows cursor naturally
   - No performance impact (GPU accelerated)
   - Works on all card components
