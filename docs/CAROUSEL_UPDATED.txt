⚙️ TECHNICAL IMPLEMENTATION
================================================================================

STATE:
- currentIndex: Tracks the first card of current page (0, 3, or 6)
- Advances by 3 (cardsPerPage)
- Loops back to 0 when reaching end

VISIBLE FEATURES:
```javascript
const visibleFeatures = features.slice(currentIndex, currentIndex + 3);
```
This gets exactly 3 consecutive cards from the array.

PAGE CALCULATION:
```javascript
const currentPage = Math.floor(currentIndex / cardsPerPage);
// Index 0-2 → Page 0
// Index 3-5 → Page 1
// Index 6-8 → Page 2
```

NAVIGATION:
```javascript
handleNext: 
  nextIndex = currentIndex + 3
  if (nextIndex >= 9) reset to 0

handlePrev:
  prevIndex = currentIndex - 3
  if (prevIndex < 0) go to last page (6)
```
