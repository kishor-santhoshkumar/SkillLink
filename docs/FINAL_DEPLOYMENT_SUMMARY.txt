API ENDPOINTS (6 Total)
================================================================================

GET    /verification/workers/{id}/badges
       → Get all badges for a worker

PATCH  /verification/workers/{id}/email
       → Verify email (admin only)

PATCH  /verification/workers/{id}/phone
       → Verify phone (admin only)

PATCH  /verification/workers/{id}/identity
       → Verify identity (admin only)

PATCH  /verification/workers/{id}/background
       → Verify background (admin only)

PATCH  /verification/workers/{id}/revoke/{type}
       → Revoke a badge (admin only)
