================================================================================
N8N EMAIL FIX - SUMMARY
================================================================================

ISSUE: Email is static/hardcoded instead of using dynamic data from webhook
CAUSE: n8n Gmail node is not configured to use webhook variables
SOLUTION: Update Gmail node to use {{ $json.field_name }} syntax

================================================================================
THE PROBLEM
================================================================================

Current Email (WRONG):
┌─────────────────────────────────────────────────────────────┐
│ To: hardcoded@example.com                                   │
│ Subject: New Job Application                                │
│ Body:                                                       │
│   A worker has applied for your job on SkillLink.           │
│   Worker Name: KISHOR.S                                     │
│   Trade: mechanic                                           │
│   Experience: 5-10 years                                    │
│   Location: Pondicherry, villianur                          │
│   Job Title: Plumber                                        │
│   Please log in to SkillLink to review the worker profile.  │
│   Regards, SkillLink Team                                   │
└─────────────────────────────────────────────────────────────┘

Issues:
❌ Email goes to hardcoded address (not company email)
❌ Subject is static (doesn't include job title)
❌ Body has hardcoded worker details (always KISHOR.S)
❌ Not personalized per company
❌ Not personalized per worker

================================================================================
THE SOLUTION
================================================================================

Updated Email (CORRECT):
┌─────────────────────────────────────────────────────────────┐
│ To: {{ $json.company_email }}                               │
│ Subject: New Job Application - {{ $json.job_title }}        │
│          Position                                           │
│ Body:                                                       │
│   Hello {{ $json.company_name }},                           │
│                                                              │
│   A worker has applied for your job on SkillLink.           │
│                                                              │
│   Worker Details:                                           │
│   Name: {{ $json.worker_name }}                             │
│   Email: {{ $json.worker_email }}                           │
│   Trade: {{ $json.trade }}                                  │
│   Experience: {{ $json.experience }}                        │
│   Location: {{ $json.location }}                            │
│                                                              │
│   Job Details:                                              │
│   Position: {{ $json.job_title }}                           │
│   Company: {{ $json.company_name }}                         │
│   Location: {{ $json.company_location }}                    │
│                                                              │
│   Please log in to SkillLink to review the application.     │
│   Regards, SkillLink Team                                   │
└─────────────────────────────────────────────────────────────┘

Benefits:
✅ Email goes to correct company (dynamic)
✅ Subject includes job title (dynamic)
✅ Body includes actual worker details (dynamic)
✅ Fully personalized per company
✅ Fully personalized per worker

================================================================================
WHAT NEEDS TO BE DONE
================================================================================

In n8n Gmail node, update 3 fields:

1. TO FIELD
   From: hardcoded@example.com
   To: {{ $json.company_email }}

2. SUBJECT FIELD
   From: "New Job Application"
   To: "New Job Application - {{ $json.job_title }} Position"

3. BODY FIELD
   From: Hardcoded HTML with static values
   To: HTML template with {{ $json.variable_name }} syntax

Time Required: 5 minutes
Difficulty: Easy
Impact: High (fixes email personalization)

================================================================================
STEP-BY-STEP INSTRUCTIONS
================================================================================

1. Open n8n workflow: worker-application

IMPORTANT NOTES
================================================================================

1. BACKEND IS CORRECT
   ✅ Backend is sending correct dynamic data
   ✅ Webhook payload includes all variables
   ✅ No changes needed to backend

2. N8N CONFIGURATION IS THE ISSUE
   ❌ Gmail node is not using webhook variables
   ❌ Gmail node has hardcoded values
   ❌ Need to update Gmail node configuration

3. VARIABLE SYNTAX
   ✅ Use {{ $json.field_name }} (double curly braces)
   ❌ Don't use $json.field_name (single braces)
   ❌ Don't use hardcoded values

4. FIELD NAMES
   ✅ Use exact field names from webhook payload
   ✅ worker_name, worker_email, trade, etc.
   ❌ Don't use workerName or worker_Name
