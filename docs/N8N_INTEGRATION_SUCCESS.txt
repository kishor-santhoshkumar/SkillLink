================================================================================
HOW IT WORKS
================================================================================

When a worker applies to a job:

1. Worker clicks "Apply Now" button
   ↓
2. Backend saves application to database
   ↓
3. Backend creates in-app notification for company
   ↓
4. Backend triggers n8n webhook with worker details
   ↓
5. n8n receives data and sends email to company
   ↓
6. Backend logs webhook response (success/failure)
   ↓
7. Frontend shows success message to worker

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Webhook returns 404
Solution: Make sure n8n workflow is ACTIVE (toggle ON)
Status: ✅ RESOLVED - Webhook is now active

Issue: Email not received
Possible causes:
- Check spam/junk folder
- Verify company user has valid email address
- Check Gmail credentials in n8n workflow
- Check n8n workflow execution logs

Issue: Webhook timeout
Possible causes:
- n8n workflow taking too long (>10 seconds)
- Network connectivity issues
- Check n8n workflow for slow nodes

Issue: Application not saved
This should NEVER happen - applications are saved BEFORE webhook
If this occurs, check backend logs for database errors
